# La Charrúa Alfajores

Handcrafted Uruguayan alfajores, made in California.

## About

La Charrúa is a small artisan alfajor business based in the Bay Area. We bring the taste of authentic Uruguayan alfajores — soft cornstarch cookies filled with slow-cooked dulce de leche, coated in chocolate, powdered sugar, or toasted coconut.

## Website

This is the source code for [lacharruaalfajores.com](https://github.com/valesanabria/LaCharruaWeb). It's a single-page static site built with pure HTML and CSS — no frameworks, no JavaScript, no build tools.

### Features

- Responsive design (desktop + mobile)
- CSS-only animations and hover effects
- Embedded images (base64) for zero external dependencies
- Glassmorphism navigation bar
- Interactive product grid and photo gallery

### Running Locally

Open `index.html` in any browser. No server or build step required.
